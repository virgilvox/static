The signaling relay. The public default works from a normal page. Running your own router? Use ws://localhost:7330.
Heads up: inside the claude.ai preview the sandbox blocks both the relay socket and the camera, so it will say "relay unreachable." That's expected. To actually use STATIC, download this file and serve it locally:
python3 -m http.server 8000  then open  http://localhost:8000/static.html
localhost counts as a secure context, so the camera works with no HTTPS. Opening the raw file (file://) won't grant camera access in most browsers — use the localhost route.
